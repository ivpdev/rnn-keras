### Prepare environment
- install virtualenv
- activate venv
- pip install tensorflow (if does not works see Tensorflow docs how to install from URL)
- pip install keras
- pip install h5py
- create folder 'data/model'


### Run
- activate venv
- python train.py
- adjust filename in predict.py to the last from weights-improvement
- python predict.py
